🚀 使用 Hugo + PaperMod 创建并部署博客到 GitHub Pages#

本教程包含三个主要步骤：

  1. 创建博客并写入内容
  2. 编译为 HTML 静态页面
  3. 发布到 GitHub Pages

🛠️ 前置准备#

  • 安装 Hugo： 下载并安装 Hugo 官网地址

  • 安装 Git Git 下载地址

  • 注册并登录 GitHub，创建一个名为：

` 用户名.github.io

的仓库（例如：`silentorchid6.github.io`）

---

## 1️⃣ 创建 Hugo 博客并安装主题

```bash
# 创建一个博客项目
hugo new site myblog
cd myblog

# 初始化 git 仓库（可选）
git init

# 安装 PaperMod 主题
git submodule add --depth=1 https://github.com/adityatelange/hugo-PaperMod.git themes/PaperMod

# 设置主题名
echo 'theme = "PaperMod"' >> hugo.toml

2️⃣ 编辑配置文件 hugo.toml#

baseURL = "https://你的用户名.github.io/"
languageCode = "zh-cn"
title = "我的 Hugo 博客"
theme = "PaperMod"
pagination.pagerSize = 10

[params]
  author = "你的名字"
  ShowReadingTime = true
  ShowShareButtons = true
  ShowPostNavLinks = true
  ShowCodeCopyButtons = true
  ShowWordCount = true
  ShowToc = true
  defaultTheme = "auto"

  [[params.socialIcons]]
    name = "email"
    url = "mailto:you@example.com"

  [[params.socialIcons]]
    name = "github"
    url = "https://github.com/你的用户名"

[menu]
  [[menu.main]]
    name = "首页"
    url = "/"
    weight = 1

  [[menu.main]]
    name = "归档"
    url = "/archives/"
    weight = 2

3️⃣ 创建博客文章#

# 创建一篇新文章
hugo new posts/hello-world.md

打开 content/posts/hello-world.md，编辑内容如下：

---
title: "Hello World"
date: 2025-07-08
draft: false
---

欢迎来到我的 Hugo 博客！

5️⃣ 编译为 HTML 文件#

# 编译为静态站点
hugo

完成后，博客 HTML 页面会生成到 public/ 目录中。

6️⃣ 部署到 GitHub Pages#

进入 public/ 目录：#

cd public
git init
git checkout -b main
git remote add origin https://github.com/你的用户名/你的用户名.github.io.git

配置 Git 用户名与邮箱（首次使用时）：#

git config --global user.name "你的 GitHub 用户名"
git config --global user.email "你的邮箱"

添加、提交并推送：#

git add .
git commit -m "🎉 初次部署博客"
git push -f origin main

✅ 部署成功！#

现在访问你的博客地址：

https://你的用户名.github.io/
